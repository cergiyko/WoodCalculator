Wood

Приложение позволяет производить расчет кубатуры и цены пиломатериалов

Для работы необходимо внести данные о породе древесины во вкладке Виды, добавить производителей во вкладке Производители, заполнить прайслист во вкладке Цены, затем во вкладке Калькулятор произвести расчет.

По каждой позиции просчитывается обем, определяется количество в 1м3, цена.

Внизу таблици подсчитывается общая стоимость всех позиций.

Также предусмотрены CRUD функции для всех сущностей
